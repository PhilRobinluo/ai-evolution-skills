# Repository Rules

This is a public skill-sharing repository.

- Do not commit private local paths, credentials, API keys, personal databases, WeChat file paths, Notion IDs, or user-specific source documents.
- Keep skills productized: clear `SKILL.md`, reusable defaults, examples, and safety boundaries.
- Prefer one high-quality skill over many half-finished skill dumps.
- Public examples should use small synthetic snippets or clearly redistributable sources.
- If a workflow needs local machine paths, express them as configurable placeholders such as `<workspace>` or `~/work/bilingual-readers`.
