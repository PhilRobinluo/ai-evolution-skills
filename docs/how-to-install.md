# How To Install

Clone the repository:

```bash
git clone https://github.com/PhilRobinluo/ai-evolution-skills.git
cd ai-evolution-skills
```

Install a skill for Codex:

```bash
mkdir -p ~/.codex/skills
cp -R skills/bilingual-reader-maker ~/.codex/skills/
```

Install a skill for Claude Code:

```bash
mkdir -p ~/.claude/skills
cp -R skills/bilingual-reader-maker ~/.claude/skills/
```

Restart your AI coding/chat session so the new skill list is refreshed.
